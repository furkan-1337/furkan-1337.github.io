# Portfolio Website

A minimalist, premium dark-themed portfolio website designed for [furkan-1337](https://github.com/furkan-1337).

## Features
- **Modern Design**: Dark aesthetic with premium neon accents.
- **Responsive**: Fully responsive layout for all devices.
- **Animations**: Smooth scroll effects (AOS) and typing animations.
- **Dynamic**: Easy to update projects and skills.

## How to Customize

1. **Edit Content**: Open `index.html` to change text, bio, and links.
2. **Update Projects**: In `index.html`, duplicate the `.project-card` styling to add more projects.
3. **Change Colors**: Open `style.css` and modify the `:root` variables:
    ```css
    :root {
        --accent: #64ffda; /* Change this to your preferred color */
    }
    ```

## Deployment on GitHub Pages

1. Push this repository to GitHub.
2. Go to **Settings** > **Pages**.
3. Under **Source**, select `main` branch and `/root` folder.
4. Click **Save**. Your site will be live at `https://furkan-1337.github.io/`.
